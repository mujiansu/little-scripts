# little-scripts
======

## INTRODUCTION
little-scripts is a collection of scripts.


## INSTALLATION
```
$ pip install -r ./requirements.txt
$ ./[scriptname].py --startup auto install
```

## Getting Started
### start
```
$ ./[scriptname].py start
```

### stop
```
$ ./[scriptname].py stop
```

### remove
```
$ ./[scriptname].py remove
```
